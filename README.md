# Personal Course Organizer Website

This project is a multi-page website developed using HTML and CSS as part of an Internet Technology course assignment.

## 📌 Features
- 4 main pages:
  - Home
  - My Courses
  - Add Course
  - Contact Us
- Organized and semantic HTML structure
- External CSS for styling
- Unified color theme across all pages
- Styled navigation bar, tables, and forms
- Responsive design implementation
- Hover effects and smooth transitions

## 📚 Functionality
- Display registered courses in a structured table
- Add new courses using a form
- View course details including instructor, schedule, and materials
- Access assignment files (PDFs)

## 🛠 Technologies Used
- HTML5
- CSS3

## 🌐 Deployment
The website is deployed using GitHub Pages and can be accessed online.

## 📁 Project Structure
- HTML files for each page
- External CSS file
- Media folder (images & video)
- PDF resources linked within the website

## 🎯 Purpose
This project demonstrates front-end web development skills including layout design, styling, and basic website structuring.
